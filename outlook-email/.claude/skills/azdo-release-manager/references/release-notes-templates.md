# Release Notes 範本庫

此文件包含各種 Release Notes 範本，適用於不同類型的發布。

## 使用指南

### 選擇適當的範本

1. **標準發布** - 用於常規的功能發布
2. **重大版本** - 用於包含重大變更的版本
3. **Hotfix** - 用於緊急修復
4. **功能預覽** - 用於預覽版或 beta 功能
5. **內部發布** - 用於開發/測試環境
6. **季度總結** - 用於定期的回顧報告

### 自訂範本

根據團隊需求調整範本：
- 保留核心結構
- 添加團隊特定的章節
- 調整表情符號和格式
- 整合團隊工具連結

### 自動化提示

使用 Azure DevOps API 可以自動填充：
- Work item 連結和標題
- 貢獻者列表
- Commit 和 PR 數量
- 建置和部署資訊
